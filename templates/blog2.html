02 Objectives

17/01/21

The primary aim of this project will be to examine how a powerchair can be better designed to accommodate the user. This will be done by looking at ways in which a chair can be made more ergonomic for someone with tetraplegia. Tetraplegia can often cause the body to become either flaccid or spastic, meaning using a regular chair can sometimes be difficult and cause incedental problems. The secondary aim is to develop a design that is more aesthetically pleasing than currently available designs.

The projects objective will be to examine existing designs with the intention of redesigning the seat to better accomodate someone with tetraplegia. Models will be produced using CAD (Computer Aided Design) software which will then be tested using in-software force analysis to replicate real-world use.

There are many categories of severity of tetraplegia making it neccessary to specify which one this project will focus on before it can proceed. This is done to specify the project, avoiding mission creep. Therefore this project will be focused on C-5 tetraplegia. This means that this means this project will focus on someone who retains some movement and strength in their arms. This project will use categorisations specified by the American Spinal Injury Associations Impairment Scale. Given this information we can now establish the projects aims...

  1. Establish the requirements of the end user and how current devices do not meet these requirements sufficiently.
  2. Produce a design concept that improves on existing designs.
  3. Produce CAD drawings that accurately depict design ideas and concluded design concept.
  4. Simulate real world usage using finite element analysis tools.
  5. Interpret the results of simulations to refine and optimise models.
